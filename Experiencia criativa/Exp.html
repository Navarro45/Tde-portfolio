Experiência Criativa

A matéria de Experiência Criativa é uma disciplina oferecida no curso de Ciência da Computação naUniversidade PUCPR. Ela abrange diversos tópicos relacionados a autodidatismo e como programar em diversas linguagens para que se possa resolver diversos tipos de problemas e tem como objetivo fornecer aos alunos a capacidade de explorar por conta própria sobre o assunto desejado e a capacidade de realizar grandes trabalhos em grupos.

Conteúdo do Curso

  • Python
  • Java
  • HTML
  • CSS

Objetivos da Matéria

  1. Capacitar os alunos no autoditatismo
  2. Capacitar os alunos para realizar trabalhos em grandes equipes
  3. Capacitar os alunos a programar em diversas linguagens de computação

Projetos Realizados

  • Projeto 1: Criação de um Jogo
  • projeto 2: Criação de um Aplicativo Multimídia
  • Projeto 3: Criação de um Website
  • Projeto 4: Criação de um Portfólio pessoal sobre o que foi realizado durante o curso
